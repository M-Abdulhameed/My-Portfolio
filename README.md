# Personal Portfolio Website

This is my personal portfolio website built entirely with HTML and CSS. It showcases my profile, skills, projects, and contact information in a clean and responsive design.

## Features
- Responsive design for desktop and mobile devices
- Modern dark-themed user interface
- About Me section
- Skills and technologies section
- Project showcase with GitHub links
- Contact and social media links
- Smooth scrolling navigation

## Technologies Used
- HTML5
- CSS3
- Font Awesome Icons
- Devicon Icons

This portfolio represents my journey as an aspiring Machine Learning Engineer and serves as a platform to showcase my projects and continuous learning in technology.
